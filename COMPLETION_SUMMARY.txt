USAGE EXAMPLES
================================================================================

New Session:
  $ dsa run data/sample.csv
  $ dsa run data/sample.csv --session-id my_experiment

With HITL + SQLite:
  $ dsa run data/sample.csv --session-id prod_run --enable-hitl --sqlite

Resume Session:
  $ dsa resume my_experiment
  $ dsa resume my_experiment --dataset data/new_dataset.csv

Monitor Sessions:
  $ dsa list-sessions
  $ dsa status my_experiment

Enable Observability:
  $ export OBSERVABILITY__LANGSMITH_ENABLED=true
  $ export OBSERVABILITY__LANGSMITH_API_KEY=ls_...
  $ dsa run data/sample.csv --session-id traced_run


NEW CONFIGURATION OPTIONS
================================================================================

Environment Variables:
  OBSERVABILITY__LANGSMITH_ENABLED=true
